Serverless - What is it? ... Use Cases

    - messages

    CI / CD

    Banking

    IoT

    Glue-logic (of BaaS)

  • Characteristics


    Ideal Serverless applications are
    latency tolerant,
    event-driven,
    short-lived

AWS Lambda

Introduced βeta in Nov 2014
Leader in developer uptake, back-end services and eco-system.

Language choices: Node.js (JS), Python, Java 8, C#, VB/F#

NEW Features - Announced at Reinvent 2017

  • NEW: Go (static binaries) and .Net(C#)

  • NEW: SAM for off-line debugging


Cost: Free tier: 1 mn req/mth, then $0.00001667/GBy-sec details
Strengths: Created the space, enormous ecosystem, Cloud9 IDE?
Weaknesses: A little slow, limited languages, lock-in

To Summarize



  • Serverless is still a new technology
  • It has a lot of promise for many workloads

  • All major cloud players are investing in the technology
  • There are many deployment choices today
  • But we need an open cloud
  • "The ultimate cloud native"
  • As platforms mature we can expect more stable performance
